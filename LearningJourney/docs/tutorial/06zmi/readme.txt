Menuri pentru ZMI
-----------------
In cadrul pachetelor din namespace-ul zope.app sunt definite cateva facilitati
pentru manipularea skinului default - denumit Rotterdam. Printre acestea sunt:

* menurile 
    * menul de adaugare, cu linkuri catre pagini de adaugare de continut in 
      folderul curent
    * menu cu paginile care pot fi accesate pentru obiectul curent (zmi_views)
* pagini autogenerate de continut


The add object menu
--------------------------------------

* Zope offers a way to create empty objects
* The browser:addItemMenu adds a new link to create a Book item
* The BookCreation class is no longer needed
